# elven-cli

A staging for build production app quickly

## dev in local

本地调试，如果 bin 已经存在，用

```
npm link
npm link --force
```

## publish

```
npm publish
```

## Install

```
npm install -g elven-cli
```

## Init a project

```
elven-cli init
```

## Install dependencies

```
yarn install
```

or

```
npm install
```

## Start project

```
yarn srtart
```

or

```
npm srtart
```
