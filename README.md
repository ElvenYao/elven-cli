# elven-cli

A staging for build production app quickly

## test in local

```
npm link
```

## publish

```
npm publish
```

## Install

```
npm install -g elven-cli
```

## Init a project

```
elven-cli init
```

## Install dependencies

```
yarn install
```

or

```
npm install
```

## Start project

```
yarn srtart
```

or

```
npm srtart
```
